                                              IMPORTANT NOTICE
                                              ----------------

    To use this application properly (In conjunction with the 'Iceberg Protocol'), only send one message
    per day, and only recieve messages one per day. The times are not important, only that this is only
    done once.

    


    (For Use In Conjuntion with the Haystack Online Module)

Instructions For Setup
-----------------------
    
    1. Clone repo into a folder of your choice
    2. Run the 'Install.sh'
    3. Run 'HaystackOnline' application

Instruction For Use
--------------------

    For Sending A Message:
 
        (After placing encrypted message on USB from 'Haystack Offline')

        1. Place encrypted message file in (*/Online/) folder

        2. Run Haystack Online, choosing 'Send'

    For Recieving Files:
    
        1.  Run HaystackOnline, choosing 'Recieve'

        2.  Move folder of files recieved onto USB

        <Continued in Hatstack Offline steps>
